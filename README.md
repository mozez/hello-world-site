# Hello World Site

A simple Hello World project with both a website and Python script.

## Files

- `index.html` - A styled Hello World webpage
- `hello.py` - A Python Hello World script

## How to Use

### View the Website

1. Open `index.html` in your browser:
   - Double-click the file, or
   - Right-click → Open with → Your browser

### Run the Python Script

1. Make sure Python is installed:
   ```bash
   python3 --version
   ```

2. Run the script:
   ```bash
   python3 hello.py
   ```

   Output:
   ```
   Hello, World!
   Welcome to my first Python project on GitHub
   ```

## Clone This Repository

git clone https://github.com/mozez/hello-world-site.git
